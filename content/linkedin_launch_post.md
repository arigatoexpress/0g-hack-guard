# LinkedIn Launch Post Draft

I built 0guard for the 0G APAC Hackathon: a pre-wallet firewall for AI agents.

The plain-English version:

An AI agent asks to do something with a wallet. 0guard checks the intent,
policy, calldata, and incident-derived risk before the wallet is ever asked to
sign. Safe simulations pass. Risky live actions stop before the signer.

The demo walks through:

- social-engineering requests that try to get an admin transfer signed;
- bridge release requests with weak verifier risk;
- proxy upgrade requests from compromised admin paths;
- safe read-only simulations that do not move funds;
- deterministic receipt hashes, Storage-ready roots, and a public 0G mainnet
  receipt anchor;
- source-aware April 2026 incident data and provenance instead of mock claims;
- 28/28 incidents source-linked and 28/28 incident-derived patterns matched by
  deterministic detectors after the Quant EIP-7702 batch-call pattern was
  promoted from public evidence.
